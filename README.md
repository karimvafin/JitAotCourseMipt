# JIT and AOT Compilers in VM (MIPT Course)

## Сборка проекта
Для сборки проекта из корневой директории следует выполнить команды
```
cmake -S . -B build
cd build && make
```

## Запуск тестов
Для запуска тестов из директории build следует выполнить команду
```
./tests/run_testNameOfTest
```
